# Project Memory

## Core
Art In: Premium streetwear for 16-30. React, Tailwind, Framer Motion, Supabase.
Typography: Outfit (display), Space Grotesk (body). Gold accent color.
Light mode uses cream/warm-grey (no pure white). Dark mode uses black/neutral.
Footer must include "Developed by Foisal Hasan Sifat" below copyright.
Dynamic DB products use `db-` ID prefix to properly merge with static catalog.
Landing-page copy/images & featured product flags are admin-editable via CMS — never hardcode in components without a fallback path.
